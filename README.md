# Form Design & Validation System | DecodeLabs Project 4

A professional and responsive web-based **Form Design & Validation System** built using **HTML, CSS, and JavaScript**.

This project focuses on creating a modern user registration form with client-side validation, dynamic error handling, and interactive user feedback. It demonstrates JavaScript DOM manipulation, event handling, and Regular Expression (Regex) validation techniques in a clean and user-friendly interface.

## Features

- 📝 Modern and responsive form design
- 👤 Full name validation with required field checking
- 📧 Email format validation
- 📱 Phone number validation
- 🔐 Strong password validation system
- 🔄 Confirm password matching
- ⚠️ Real-time error message display
- ✅ Success message after valid submission
- 🎨 Modern UI design with smooth input effects
- 📱 Fully responsive layout for all devices

## Technologies Used

- HTML5
- CSS3 (Responsive Design + Modern UI Styling)
- JavaScript (DOM Manipulation + Event Handling + Regex Validation)

## How It Works

- When a user fills the form and clicks **Submit**:

  - Form submission is controlled using JavaScript
  - All input fields are validated automatically
  - Invalid data shows relevant error messages
  - Correct inputs receive validation feedback
  - A success message appears after successful validation

- The validation system checks:

  - Empty fields
  - Invalid email formats
  - Incorrect phone numbers
  - Weak passwords
  - Password mismatch
